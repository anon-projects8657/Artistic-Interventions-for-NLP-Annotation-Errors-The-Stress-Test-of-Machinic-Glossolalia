# Artistic Interventions for NLP Annotation Challenges: The Stress Test of Machinic Glossolalia
Mother Board’s Mother Tongue is a computational linguistics and artistic research project that explores a machine’s vocal production of glossolalia-like language. Glossolalia is colloquially known as ‘speaking in tongues.’ The linguistic phenomenon consists of the human production of seemingly unintelligible utterances. It is, by its nature, difficult to annotate accurately within the purview of Natural Language Processing (NLP). This project attempts to creatively repurpose the inherent annotation errors of glossolalia. The glossolalia-producing machinic system demonstrated here consists of the interaction of 1) a ‘nonsense’ linguistic corpus – curated from transcripts of practitioners and publicly available corpora, 2) a micro-controller based environmental data stream inputting into the system and 3) a fine-tuned large language model executable through Python. While discussing some philosophical considerations of machinic glossolalia, we also address some methodological considerations for NLP. Ultimately, using the artistic project as a case study, we argue that machinic glossolalia presents a stress test for linguistic definitions like ‘linguistic data,’ what can constitute a ‘corpus,’ the assessment of seemingly nonsensical machinic output, and how some performance art precedents could inform creative uses of NLP methods.
